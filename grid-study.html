naming

Grid lines are automatically assigned positive numbers ...

grid-template-columns:
[c1] 1fr [c2] 2fr [c3] 1fr [c4] 2fr [c5];
grid-template-rows:
[r1] 1fr [r2] 2fr [r3] 1fr [r4];

...but they can be named also.

lines can be referenced by their name and count simultaneously

column/row - start/end

item-1 {
grid-row-start: 1;
grid-row-end: 4;
grid-column-start: 1;
grid-column-end: 2; }
item-2 {
grid-row-start: r3;
grid-row-end: r4;
grid-column-start: c2;
grid-column-end: c4; }
grid-row: start line / end line (or span value)

grid-column: start line / end line (or span value)

shorthand

item-3 {
grid-row: 1 / 3;
grid-column: 2 / 3; }
item-4 {
grid-row: r1 / span 1;
grid-column: c3 / span 2; }

example.

span [number]

the item will span across the provided number of grid tracks

span [name]

span across the next line with the provided name

item-5 {
grid-row: r2 / span 1;
grid-column: c3 / span c5; }

grid-area

grid-area:
grid-row-start / grid-column-start / grid-row-end / grid-column-end | itemname;

specifies a grid item's size and location. can also be used to assign a name to a grid item.

item-3 {
grid-area: 1 r1; / 2 c2; / 3 r3; / 3 c3 }
item-4 {
grid-area: 1 r1; / 4 c4; / 2 r2; / 5 c5; }
item-5 {
grid-area: 2 r2; / 3 c3; / 3 r3; / 4 c4; }

implicit tracks

grid-auto-columns: | grid-auto-rows:

Implicit tracks get created when there are more grid items than cells in the grid, or when a grid item is placed outside of the explicit grid.
